# Targoviste-2.0
Rewrite of Targoviste archiving library.
Key features:
 * Written in simple, pure and ethnically clean ANSI C.
 * Has very simple api
 * Portable.
 * C89 compiliant
 * Very well documented
 * Keeping so-called "C climate"
